PXE方式安装系统

  1. 重启单板至UEFI主菜单，在“Boot Manager”菜单中选择从“EFI Network 2”启动。
  2. 启动至Grub菜单，选择相应的安装启动项，例如“Install CentOS Linux 7 D03 –network”。
  3. 系统将实现自动安装。
